# Privacy-First Applications

End-to-end encrypted applications that put user privacy and data sovereignty first. These tools provide practical alternatives to surveillance-based services while maintaining usability and feature parity.

## Cloud Storage & Productivity

### Ente
[ente-io/ente](https://github.com/ente-io/ente) - Fully open source, end-to-end encrypted cloud for everything. The monorepo contains **Ente Photos** (alternative to Apple and Google Photos with 3x data replication, face detection, semantic search, private sharing, and collaborative albums) and **Ente Auth** (2FA alternative to the deprecated Authy). Built with Dart, offers 10GB free storage and supports self-hosting. Externally audited by Cure53, Symbolic Software, and Fallible. Available on iOS, Android, F-Droid, Web, Linux, macOS, and Windows.

### Anytype
[anyproto/anytype-api](https://github.com/anyproto/anytype-api) - Developer portal for extending Anytype through powerful integrations. Anytype is a privacy-focused, local-first knowledge management platform. The API enables global & space search, workspace management, object & list organization, and custom types & templates. Includes Raycast extension ([anytype-raycast](https://github.com/anyproto/anytype-raycast)) for Mac and MCP server ([anytype-mcp](https://github.com/anyproto/anytype-mcp)) enabling AI assistants to interact with Anytype through natural language.

## Password Management

### PassForgePro
[0DDW/PassForgePro](https://github.com/0DDW/PassForgePro) - Zero-knowledge password vault that keeps credentials safe using advanced encryption on your device. Built with Python, provides secure local storage without trusting third-party servers. All encryption happens client-side ensuring the service provider never has access to plaintext passwords.

### Vaulton
[Johncraz/vaulton](https://github.com/Johncraz/vaulton) - Open-source, self-hosted password manager with end-to-end encryption. Built with TypeScript, offers complete control over your password data with the security of modern encryption standards. Alternative to commercial password managers with full data sovereignty.

### Nostr Web of Trust
- [nostr-wot/nostr-wot](https://github.com/nostr-wot/nostr-wot) - Nostr Web of Trust toolkit: query social graph distance between pubkeys, includes browser extension and oracle API
- [nostr-wot/nostr-wot-extension](https://github.com/nostr-wot/nostr-wot-extension) - Browser extension for Nostr WoT: query custom trust scores and social distances
- [barrydeen/wot-relay](https://github.com/barrydeen/wot-relay) - Archive every note in your Nostr web of trust (82 stars)
- [Pretty-Good-Freedom-Tech/brainstorm](https://github.com/Pretty-Good-Freedom-Tech/brainstorm) - Personalized Nostr relay featuring the Grapevine web of trust

### General Web of Trust
- [flammafex/clout](https://github.com/flammafex/clout) - Censorship-resistant social protocol built on Web of Trust; feed filtered by trust distance, not engagement algorithms
- [alpeware/bitecho](https://github.com/alpeware/bitecho) - Decentralized WebRTC signaling service and cryptographic Web of Trust for peer-to-peer agents
- [dtdannen/arrival](https://github.com/dtdannen/arrival) - Anonymous, verifiable reviews based on your Web of Trust
- [flox1an/almond](https://github.com/flox1an/almond) - Temporary BLOSSOM file storage service with Nostr-based authorization and web of trust support

### GPG Web of Trust
- [mricon/wotmate](https://github.com/mricon/wotmate) - Web of trust grapher and analyzer (40 stars)

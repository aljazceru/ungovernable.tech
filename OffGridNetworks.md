#### LoRa Mesh Networks
- [Meshtastic](https://meshtastic.org/) - open source, off-grid, decentralized mesh network using LoRa
- [Meshtastic GitHub](https://github.com/meshtastic)
- [Meshtastic firmware](https://github.com/meshtastic/firmware)
- [MeshTastic Android](https://github.com/meshtastic/Meshtastic-Android)
- [Ripple](https://github.com/ripple-lib/ripple) - experimental LoRa mesh networking

#### Reticulum Stack
- [Reticulum](https://reticulum.network/) - self-configuring, encrypted mesh networking stack
- [Reticulum GitHub](https://github.com/markqvist/Reticulum)
- [NomadNet](https://github.com/markqvist/NomadNet) - encrypted communications network built on Reticulum
- [Sideband](https://github.com/markqvist/Sideband) - LXMF client for Android, Linux and macOS
- [LXMF](https://github.com/markqvist/LXMF) - secure, distributed messaging protocol
- [RNode](https://github.com/markqvist/RNode_Firmware) - open LoRa radio hardware

#### MeshCore
- [MeshCore](https://github.com/ajmcquilkin/meshcore) - Meshtastic-like mesh networking for T-Deck, T-Watch, and other ESP32 devices
- [MeshCore Radio](https://github.com/ajmcquilkin/meshcore-radio) - radio firmware component

#### Delay-Tolerant Networks (DTN)
- [Briar](https://briarproject.org/) - secure messaging via Bluetooth, WiFi, or Tor
- [Briar GitHub](https://code.briarproject.org/briar/briar)
- [Secure Scuttlebutt](https://ssbc.github.io/scuttlebutt-protocol-guide/) - P2P gossip protocol
- [Manyverse](https://www.manyver.se/) - social network using SSB

#### Overlay Mesh Networks
- [Yggdrasil](https://yggdrasil-network.github.io/) - overlay network implementing a greedy routing scheme
- [Yggdrasil GitHub](https://github.com/yggdrasil-network/yggdrasil-go)
- [cjdns](https://github.com/cjdelisle/cjdns) - encrypted IPv6 mesh network
- [Hyperboria](https://hyperboria.net/) - cjdns-based mesh network

#### Decentralized P2P
- [GNUnet](https://gnunet.org/) - framework for secure, decentralized networking
- [GNUnet GitHub](https://github.com/NicolasL-S/gnunet)
- [libp2p](https://libp2p.io/) - modular networking stack for P2P applications
- [libp2p GitHub](https://github.com/libp2p/go-libp2p)
- [ipfs-cluster](https://github.com/ipfs-cluster/ipfs-cluster) - replicate datasets across IPFS
